# M3DocRAG Experimental Notebooks

This folder contains a collection of Jupyter notebooks used for experiments with M3DocRAG and baseline MoLoRAG implementations.

## Notebooks Overview

- **`01_baseline_experiment.ipynb`**: Initial baseline experiments for MoLoRAG.
- **`02_local_baseline_experiment.ipynb`**: Local testing and validation of the baseline model.
- **`03_m3docrag_baseline_experiment.ipynb`**: Specific experiments focused on M3DocRAG integration.
- **`MoLoRAG_Reproduction_Final.ipynb`**: Final consolidated reproduction notebook for project reporting.
- **`textRag.ipynb`**: Experiments with text-based RAG implementations.

## Usage

These notebooks can be run using Jupyter or VS Code's Interactive Window. Ensure all dependencies listed in the root `requirements.txt` are installed in your environment.
